# たねまき・とけい

対話型算数スクール「たねまき」の授業用シミュレーター。先生が画面共有しながら「○じ・○じはん」を見せるためのツールです。

🔗 **使ってみる**: https://pokopy-talkpapa.github.io/tanemaki-tokei/

## 使い方

1. 「なんじ」のプルダウンで時間（1〜12）を選ぶ
2. 「ふん」のプルダウンで「ちょうど」か「はん」を選ぶ
3. 選んだ瞬間に針がアニメーションで動きます
4. 下に「○じ ちょうど」「○じ はん」が大きく表示されます

## 特徴

- **ヨーメル式文字盤**：内側に「○じ」エリアがあり、短針の先端が指し示す時刻がひと目で分かります
- **常に時計回り**：戻すときも一周回ってリアル時計と同じ動き
- **長針と短針が連動**：長針半周＝短針15°ぴったり

## 想定シーン

オンライン授業（スクールタクト併用）で、先生がブラウザでこのページを画面共有 → 子どもは見ながら時計の仕組みを理解。
